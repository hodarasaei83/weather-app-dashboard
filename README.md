#Weather Dashboard

A modern, responsive weather dashboard built with React, TypeScript, and Vite. This application provides real-time weather information with beautiful visualizations and an intuitive user interface.

##Features

- **Real-time Weather Data**: Get current weather, forecasts, and detailed metrics
- **Interactive Dashboard**: Visualize weather trends with Recharts graphs
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Multiple Locations**: Search and save favorite cities
- **Dark/Light Mode**: Toggle between themes

##Tech Stack

- **React** - Frontend library
- **TypeScript** - Type safety
- **Vite** - Build tool and development server
- **shadcn/ui** - UI component library
- **Recharts** - Data visualization
- **TanStack Query (React Query)** - Data fetching and state management
- **Weather API** - [Open weather map] -**npm** - Package manager

### Installation

1. **Clone the repository**

   ```bash
   git clone <https://github.com/hodarasaei83/weather-app-dashboard>

   ```

2. **Install dependencies**

```bash
npm install
```

3. **Set up environment variables**

add your Weather Api key to .env file
VITE_OPENWEATHER_API_KEY=your_api_key

4. **Start the dev server**

```bash
npm run dev
```
